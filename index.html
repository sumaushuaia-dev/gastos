Escaneo de facturas con IA

Para que la app pueda leer fotos de facturas/tickets necesitás tu propia clave (API key) de la API de Anthropic. Se guarda solo en este dispositivo, en tu navegador, nunca se envía a ningún otro lado más que a Anthropic.

Podés crear una clave en console.anthropic.com → API Keys. La API de Anthropic cobra por uso; el costo de leer una foto de ticket es muy bajo (centavos de dólar).

Copia de seguridad

Los datos se guardan en este navegador/dispositivo. Si vas a usar la app en el celular y en la compu, exportá los datos en uno y "Importar" en el otro para tenerlos sincronizados.
